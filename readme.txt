== Who is this for? ==

Are you looking to create resumes that stand out? Want to be part of the video revolution? Then this plugin is for you.

== Installation ==
 
1. Upload plugin zip file to your plugins directory. Usually that would be to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That is it or
1. Use the Plugins Add new section to find the plugin and install
 
== Frequently Asked Questions ==

= How does it work? =

The plugin combines and extends the Resume Manager addon of the Job Manager plugin together with the Ziggeo core plugin. This brings the best of both worlds into a single system that you can build upon.

With this combination you would make your service utilize the best of both worlds. All that and without any codding, just install, activate and enjoy.
